DROP YOUR LOGO FILE HERE
=========================

When your final logo is ready, save it into this folder using ONE of these
exact filenames (match the format of your file):

  logo.svg   ← preferred if you have a vector/SVG version (looks crisp at any size)
  logo.png   ← use this if you only have a raster file (PNG with a transparent background works best)

That's it — just this one file, named exactly "logo.svg" or "logo.png",
directly inside this assets/ folder (not in a subfolder).

Then, in Claude Code, use the Phase 5 prompt from the deployment plan
(babyaquaspa-deployment-plan.md) and it will find this file automatically
and wire it into all 5 pages for you.

Ideal logo specs, if you have any control over how it's exported:
- Square or circular crop (it sits inside a round badge in the header/footer)
- At least 200x200px if PNG (SVG has no size limit — always sharp)
- Transparent background preferred, since the badge behind it changes
  color between the navy footer and the header
